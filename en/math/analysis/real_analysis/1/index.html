    测度论

    一、基本定义与符号约定

    • \(\mathbb{R}^n\) 表示 \(n\) 维欧几里得空间。
    • 点集 \(E\), \(E_1\), \(E_2, \dots\) 均为 \(\mathbb{R}^n\) 的子集。若其满足 Carathéodory 条件，则称为可测集。
    • \(m(E)\) 或 \(m^*(E)\) 表示集合 \(E\) 的 Lebesgue 外测度。当 \(E\) 可测时，外测度 \(m^*(E)\) 即称为其 Lebesgue 测度，仍简记为 \(m(E)\)。
    • 给定一列递增集合 \(E_1 \subset E_2 \subset \cdots\)，定义其“差集”为： [ g_1 = E_1, \quad g_k = E_k \setminus E_{k-1} \ (k \ge 2). ] 显然，\(\{g_k\}\) 是两两不交的集合列，并且满足： [ E = \bigcup_{k=1}^\infty E_k = \bigcup_{k=1}^\infty g_k. ]

    二、可数可加性 (Countable Additivity)

    可数可加性是测度最核心的性质，它将有限运算推广至可数无穷。

    定理 1 (可数可加性)

    设 \(\{E_j\}_{j=1}^\infty\) 为一列两两不交的可测集，记 \(E = \bigcup_{j=1}^\infty E_j\)，则 \(E\) 可测，且有 [ m(E) = \sum_{j=1}^\infty m(E_j)。 ]

    证明：
    我们利用引言中定义的 \(E_k\) 和 \(g_k\) 来证明。设 \(E_k = \bigcup_{j=1}^k E_j\)，则 \(\{E_k\}\) 是递增的可测集列。定义 \(g_k\) 如引言所述，则 \(\{g_k\}\) 两两不交，且 \(E_k = \bigcup_{j=1}^k g_j\)，\(E = \bigcup_{j=1}^\infty g_j\)。

    1. 由有限可加性到可数和： 由于 \(\{g_k\}\) 两两不交，且每个 \(E_k\) 是有限个 \(g_j\) 的并，由测度的有限可加性，有： [ m(E_k) = m\left( \bigcup_{j=1}^k g_j \right) = \sum_{j=1}^k m(g_j)。 ]

    2. 证明极限关系： 一方面，因为 \(E_k \subset E\) 且测度具有单调性，有 \(\lim_{k\to\infty} m(E_k) \le m(E)\)。 另一方面，对任意 \(N \in \mathbb{N}\)，有 \(\bigcup_{j=1}^N g_j = E_N \subset E\)，所以 \(\sum_{j=1}^N m(g_j) = m(E_N) \le m(E)\)。令 \(N \to \infty\)，得 \(\sum_{j=1}^\infty m(g_j) \le m(E)\)。 结合以上两点，我们得到： [ m(E) = \sum_{j=1}^\infty m(g_j)。 ]

    3. 完成最终证明： 注意到对每个 \(j\)，有 \(E_j = g_j\)，因为 \(E_j\) 与之前的 \(E_i (i<j)\) 不交，所以 \(E_j = E_j \setminus E_{j-1} = g_j\)。因此，\(\sum_{j=1}^\infty m(g_j) = \sum_{j=1}^\infty m(E_j)\)。 代入上式，即得： [ m(E) = \sum_{j=1}^\infty m(E_j) = \lim_{k\to\infty} \sum_{j=1}^k m(E_j) = \lim_{k\to\infty} m(E_k)。 ] 最后一个等式体现了测度的“连续性”：对于递增的可测集列，其测度的极限等于其并集的测度。

    三、可测集的逼近性质 (Approximation Properties)

    可测集可以用“更规则”的集合（开集、闭集、紧集）从外部或内部以任意精度逼近。

    引理 1 (开集与闭集逼近)

    设 \(E \subset \mathbb{R}^n\) 可测，\(m(E) < \infty\)。则对任意 \(\epsilon > 0\)： 1. （外逼近）存在开集 \(G \supset E\)，使得 \(m(G \setminus E) < \epsilon\)。 2. （内逼近）存在闭集 \(F \subset E\)，使得 \(m(E \setminus F) < \epsilon\)。

    证明概要： 1. 由 Lebesgue 外测度的定义，存在一列开长方体 \(\{I_j\}\) 覆盖 \(E\)，使得 \(\sum |I_j| < m(E) + \epsilon\)。取开集 \(G = \bigcup I_j\)，则 \(E \subset G\)，且 \(m(G) \le \sum |I_j| < m(E) + \epsilon\)。由 \(G = E \cup (G \setminus E)\) 及可测集测度的可加性，有 \(m(G) = m(E) + m(G \setminus E)\)，故 \(m(G \setminus E) = m(G) - m(E) < \epsilon\)。 2. 考虑 \(E\) 的补集 \(E^c\)，应用 (1) 于 \(E^c\)，存在开集 \(G \supset E^c\) 使得 \(m(G \setminus E^c) < \epsilon\)。令 \(F = G^c\)，则 \(F\) 是闭集，且 \(F \subset E\)。又因为 \(E \setminus F = E \cap G = G \setminus E^c\)，所以 \(m(E \setminus F) = m(G \setminus E^c) < \epsilon\)。

    定理 2 (紧集逼近)

    设 \(E \subset \mathbb{R}^n\) 可测，\(m(E) < \infty\)。则对任意 \(\epsilon > 0\)，存在紧集 \(K \subset E\)，使得 \(m(E \setminus K) < \epsilon\)。

    证明： 1. 先用闭集内部逼近：由引理1(2)，存在闭集 \(F \subset E\)，使得 \(m(E \setminus F) < \frac{\epsilon}{2}\)。 2. 将无界闭集截断为紧集：考虑一列半径递增的闭球 \(\overline{B_n(0)}\)。定义 \(K_n = F \cap \overline{B_n(0)}\)。显然，每个 \(K_n\) 是闭集且有界（从而是紧集），并且 \(K_1 \subset K_2 \subset \cdots\)，且 \(\bigcup_{n=1}^\infty K_n = F\)。 3. 利用测度连续性：由定理1的结论（递增集列的极限），有 \(\lim_{n\to\infty} m(K_n) = m(F)\)。因此，存在某个充分大的 \(N\)，使得 \(m(F) - m(K_N) < \frac{\epsilon}{2}\)，即 \(m(F \setminus K_N) < \frac{\epsilon}{2}\)。 4. 得到最终估计：取 \(K = K_N\)，它是紧集且 \(K \subset E\)。那么， [ m(E \setminus K) = m((E \setminus F) \cup (F \setminus K)) \le m(E \setminus F) + m(F \setminus K) < \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon。 ] 这就完成了证明。

    注：此定理表明，对于测度有限的可测集，其内部的“有界性”和“闭性”可以同时被很好地控制。当 \(E\) 有界时，证明可以更直接。

    四、次可数可加性 (Countable Subadditivity)

    次可数可加性是外测度（不仅仅是测度）的基本性质，对任意集合列都成立。

    定理 3 (外测度的次可数可加性)

    对 \(\mathbb{R}^n\) 中任意一列集合 \(\{E_j\}_{j=1}^\infty\)（不一定可测，也不一定不交），有 [ m^\left( \bigcup_{j=1}^\infty E_j \right) \le \sum_{j=1}^\infty m^(E_j)。 ]

    证明： 如果某个 \(m^*(E_j) = +\infty\)，不等式显然成立。现假设所有 \(m^*(E_j) < +\infty\)。

    由外测度的定义（下确界），对每个 \(j\) 和任意 \(\epsilon > 0\)，存在一列开长方体 \(\{I_j^{(k)}\}_{k=1}^\infty\) 覆盖 \(E_j\)，使得 [ \sum_{k=1}^\infty |I_j^{(k)}| < m^(E_j) + \frac{\epsilon}{2^j}。 ] 考虑所有这些长方体的全体 \(\{I_j^{(k)}\}_{j,k=1}^\infty\)，它构成了 \(\bigcup_{j=1}^\infty E_j\) 的一个覆盖。因此，由外测度的定义，有 [ m^\left( \bigcup_{j=1}^\infty E_j \right) \le \sum_{j=1}^\infty \sum_{k=1}^\infty |I_j^{(k)}| < \sum_{j=1}^\infty \left( m^(E_j) + \frac{\epsilon}{2^j} \right) = \left( \sum_{j=1}^\infty m^(E_j) \right) + \epsilon。 ] 由于 \(\epsilon\) 是任意正数，令 \(\epsilon \to 0\) 即得所需不等式。

    五、重要推论及其证明

    由以上基本定理，我们可以推导出一系列关于测度运算的关键性质。

    推论 1 (单调性)

    若 \(E_1\), \(E_2\) 可测，且 \(E_1 \subset E_2\)，则 \(m(E_1) \le m(E_2)\)。若进一步有 \(m(E_2) < \infty\)，则 \(m(E_2 \setminus E_1) = m(E_2) - m(E_1)\)。

    证明：
    因为 \(E_1\) 和 \(E_2 \setminus E_1\) 不交，且 \(E_2 = E_1 \cup (E_2 \setminus E_1)\)，由可数可加性（此处退化为有限可加性）有 \(m(E_2) = m(E_1) + m(E_2 \setminus E_1)\)。由于测度非负，\(m(E_2 \setminus E_1) \ge 0\)，故 \(m(E_1) \le m(E_2)\)。当 \(m(E_2) < \infty\) 时，移项即得 \(m(E_2 \setminus E_1) = m(E_2) - m(E_1)\)。

    推论 2 (可测集的运算封闭性)

    可测集的有限并、可数并、有限交、可数交、差集和补集均为可测集。

    证明思路：
    由 Carathéodory 条件与集合的运算性质可以证明。特别地，\(E\) 可测蕴含其补集 \(E^c\) 可测。这是可测集构成一个 \(\sigma\)-代数的基础。

    推论 3 (递增集列的连续性)

    设 \(\{E_j\}_{j=1}^\infty\) 为一列递增的可测集（\(E_1 \subset E_2 \subset \cdots\)），记 \(E = \bigcup_{j=1}^\infty E_j\)，则有 [ m(E) = \lim_{j \to \infty} m(E_j)。 ]

    证明：
    这正是定理1证明过程的直接结论。通过定义 \(g_k = E_k \setminus E_{k-1}\)，利用可数可加性得到 \(m(E) = \sum_{k=1}^\infty m(g_k) = \lim_{n\to\infty} \sum_{k=1}^n m(g_k) = \lim_{n\to\infty} m(E_n)\)。

    推论 4 (递减集列的连续性)

    设 \(\{E_j\}_{j=1}^\infty\) 为一列递减的可测集（\(E_1 \supset E_2 \supset \cdots\)），且 \(m(E_1) < \infty\)，记 \(E = \bigcap_{j=1}^\infty E_j\)，则有 [ m(E) = \lim_{j \to \infty} m(E_j)。 ]

    证明：
    考虑集合列 \(F_j = E_1 \setminus E_j\)。显然，\(\{F_j\}\) 是递增的，并且 \(\bigcup_{j=1}^\infty F_j = E_1 \setminus E\)。由推论3，有 \(\lim_{j\to\infty} m(F_j) = m(E_1 \setminus E)\)。又因为 \(E_1 = E_j \cup F_j\) 且 \(E_j\) 与 \(F_j\) 不交，由有限可加性有 \(m(E_1) = m(E_j) + m(F_j)\)。同理，\(m(E_1) = m(E) + m(E_1 \setminus E)\)。两式相减并结合极限，得到 \(\lim_{j\to\infty} m(E_j) - m(E) = 0\)，即 \(\lim_{j\to\infty} m(E_j) = m(E)\)。

    注：条件 \(m(E_1) < \infty\) 是必要的，否则可能出现 \(\infty - \infty\) 的不定式。

    推论 5 (有限并逼近无限并)

    设 \(\{E_j\}_{j=1}^\infty\) 为一列两两不交的可测集。记部分并 \(K_N = \bigcup_{j=1}^N E_j\)，则有 [ \lim_{N \to \infty} m(K_N) = m\left( \bigcup_{j=1}^\infty E_j \right)。 ] 即，无限不交并的测度可以通过有限不交并的测度来逼近。

    证明：
    这是可数可加性的直接结果：\(m\left( \bigcup_{j=1}^\infty E_j \right) = \sum_{j=1}^\infty m(E_j) = \lim_{N\to\infty} \sum_{j=1}^N m(E_j) = \lim_{N\to\infty} m(K_N)\)。

    推论 6 (递增有界可测集的逼近)

    设 \(E\) 可测。取一列递增的紧集或有界可测集 \(\{B_j\}\) 使得 \(\bigcup_{j=1}^\infty B_j = \mathbb{R}^n\)（例如 \(B_j\) 为半径为 \(j\) 的闭球）。定义 \(E_j = E \cap B_j\)，则 \(\{E_j\}\) 是递增的有界可测集列，且 \(E = \bigcup_{j=1}^\infty E_j\)。由推论3，有 [ m(E) = \lim_{j \to \infty} m(E_j)。 ] 这个性质在将一般可测集上的问题化归为有界可测集问题时非常有用。
