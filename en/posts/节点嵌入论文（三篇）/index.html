    图嵌入方法：DeepWalk、node2vec 与 NetMF

    围绕基于随机游走的图表示学习，介绍三种紧密关联的方法：DeepWalk、node2vec 以及将它们统一为矩阵分解的 NetMF。目标是将图中的节点映射到低维向量空间，使向量之间的相似性能够反映原始图的结构特征。

    1. 图节点嵌入的基本框架

    许多图机器学习任务需要将节点表示为固定维度的实向量。理想情况下，这些向量应当保持图中节点间的邻域关系。形式化地说，我们希望学习一个映射 [ f: v \in V \rightarrow \mathbb{R}^d ] 使得相似节点在向量空间中彼此接近。

    早期方法依赖手工构造的特征，例如节点度数、PageRank 值、聚类系数或谱嵌入。这些方法表达能力有限，迁移到大规模图时面临困难。基于随机游走的方法通过自动采样图结构并将其输入语言模型，实现了更灵活的表征学习。

    一个共同的建模思路是将问题表述为最大化邻域概率。给定节点 \(u\) 的向量表示 \(f(u)\)，我们希望它能很好地“解释”其邻域节点集合 \(NS(u)\)，即最大化 [ \max_f \sum_{u \in V} \log P(NS(u) \mid f(u)). ] 为了简化模型，通常引入两个假设：

    • 条件独立性：给定中心节点 \(u\)，其邻域中各节点的出现相互独立。 [ P(NS(u) \mid f(u)) = \prod_{n_i \in NS(u)} P(n_i \mid f(u)). ]
    • 对称性与 softmax 建模：节点间的关系用向量内积表示相似度，并通过 softmax 转化为概率。 [ P(n_i \mid f(u)) = \frac{\exp(f(n_i) \cdot f(u))}{\sum_{v \in V} \exp(f(v) \cdot f(u))}. ]

    代入后得到的优化目标为 [ \max_f \sum_{u \in V} \left[ - \log Z_u + \sum_{n_i \in NS(u)} f(n_i) \cdot f(u) \right], ] 其中归一化项 \(Z_u = \sum_{v \in V} \exp(f(v) \cdot f(u))\)。直接计算该归一化项需要遍历全部 \(|V|\) 个节点，代价极高。实际优化中采用负采样（negative sampling） 来近似，每次只采样少量负样本，避免对全节点求和。这一技巧来自 word2vec，也是后续方法可扩展的关键。

    2. DeepWalk：把图当作语言模型

    DeepWalk 的核心思想是将图中的随机游走序列视为自然语言中的句子，把节点当作单词，然后借助 Skip-gram 模型学习节点的向量表示。

    2.1 随机游走生成语料

    从图中每个节点开始，进行若干次随机游走。每次游走从起点 \(u\) 开始，每一步均匀随机地选择一个邻居并前进，游走长度固定为 \(l\)。这样得到的一条路径 [ c_0 = u, c_1, c_2, \dots, c_l ] 称为一条随机游走序列。对每个节点重复 \(r\) 次，收集得到的所有序列构成一个“语料库”。这个过程相当于对图的局部结构进行采样：经常共现在同一条游走路径中的节点，倾向于在图中处于相近的社区或扮演相似的角色。

    2.2 Skip-gram 学习节点表示

    Skip-gram 的目标是，给定中心节点，预测其上下文窗口内的邻居节点。在 DeepWalk 中，上下文窗口大小记为 \(k\)。对于语料中出现的每个中心节点 \(u\) 及其窗口内的邻居 \(v\)，我们希望最大化 [ \log P(v \mid u). ] 利用前面描述的 softmax 定义和负采样技术，目标函数转化为 [ \log \sigma(u \cdot v) + \sum_{j=1}^K \log \sigma(-u \cdot v_j), ] 其中 \(\sigma\) 为 sigmoid 函数，\(v_j\) 是从噪声分布中采样的负样本节点。通过随机梯度下降优化该目标，最终得到所有节点的嵌入向量。

    2.3 整体流程

    算法步骤可概括为： 1. 对每个节点 \(u\)，重复 \(r\) 次：生成长度为 \(l\) 的随机游走，加入语料。 2. 在生成的游走语料上训练 Skip-gram 模型。 3. 输出学习到的节点嵌入向量。

    DeepWalk 之所以有效，在于随机游走将高阶邻接关系展开为线性序列，而 Skip-gram 则通过共现统计将这种结构压缩到低维向量中。本质上，它是在隐式地分解一个与随机游走转移概率相关的矩阵。然而，标准的随机游走无法区分不同类型的结构相似性，这引出了后续的改进方法。

    3. node2vec：有偏随机游走与结构探索

    DeepWalk 的随机游走在选择下一步时是均匀的，相当于混合采集两种信息：同质性（相邻节点相似）和结构等价性（处于相似结构位置的节点相似）。实际图分析中，这两者对应不同的需求。例如，广度优先搜索（BFS）倾向捕捉局部邻域的同质性，而深度优先搜索（DFS）更能反映节点在全局网络中的角色。node2vec 通过设计有偏随机游走，引入两个参数来灵活控制游走行为。

    3.1 有偏转移概率

    假设当前路径片段为 \(t \rightarrow v\)，现在要从 \(v\) 决定下一步走到 \(x\)。定义未归一化的转移倾向 \(\pi_{vx}\) 如下： [ \pi_{vx} = \begin{cases} \frac{1}{p} & \text{如果 } d_{tx}=0 \;\text{(回到上一步节点 } t\text{)}\ 1 & \text{如果 } d_{tx}=1 \;\text{(走到与 } t \text{ 直接相连的节点)}\ \frac{1}{q} & \text{如果 } d_{tx}=2 \;\text{(走到与 } t \text{ 距离为2的节点)} \end{cases} ] 其中 \(d_{tx}\) 表示节点 \(t\) 和 \(x\) 之间的最短路径距离。再通过归一化得到实际转移概率： [ P(c_i = x \mid c_{i-1}=v) = \frac{\pi_{vx}}{\sum_{y \in N(v)} \pi_{vy}}. ]

    参数含义： - 返回参数 \(p\)：控制返回已访问节点的概率。\(p\) 较大时，游走不易回溯，趋向探索新区域；\(p\) 较小时，游走倾向于局部徘徊。 - 进出参数 \(q\)：控制游走是偏向 DFS 还是 BFS。\(q > 1\) 时，倾向于靠近 \(t\) 的节点，类似 BFS，捕捉同质性；\(q < 1\) 时，倾向于远离 \(t\) 的节点，类似 DFS，捕捉结构等价性。

    3.2 算法流程

    node2vec 算法的整体框架与 DeepWalk 类似，但游走生成部分使用了有偏随机游走。首先根据输入图 \(G\) 和参数 \(p,q\) 计算转移倾向 \(\pi\)，构建一个带权重的有向图 \(G'\)。然后对每个节点，以 \(G'\) 为基础生成 \(r\) 条长度为 \(l\) 的有偏游走。最后同样在游走语料上通过 Skip-gram 学习嵌入。

    在采样下一节点时，node2vec 使用别名采样（alias sampling），将离散分布的采样复杂度降至 \(O(1)\)，比传统的二分查找 \(O(\log n)\) 更快，能高效处理大规模图中大量游走的生成需求。

    3.3 从 DeepWalk 到 node2vec

    DeepWalk 可以看作 node2vec 在 \(p=1, q=1\) 时的特例。通过调整 \(p\) 和 \(q\)，node2vec 使学习到的嵌入能够偏重同质性或结构等价性，在不同类型的预测任务中表现更好。然而，这些方法背后的理论本质仍然是隐式的，直到 NetMF 将其统一为明确的矩阵分解形式。

    4. NetMF：统一为矩阵分解

    基于随机游走和 Skip-gram 的方法虽然有效，但理论解释长期缺失：为什么这些操作能够产生有意义的图嵌入？2018 年发表的 NetMF 论文给出了答案：DeepWalk、LINE、node2vec 等算法本质上都是在隐式地分解某个由图的转移概率构成的闭式矩阵。

    4.1 核心等价性

    考虑一个无向连通图 \(G=(V,E)\)，邻接矩阵 \(A\)，度矩阵 \(D\)，转移概率矩阵 \(P=D^{-1}A\)，图的体积 \(\text{vol}(G)=\sum_i d_i\)。设 Skip-gram 的上下文窗口大小为 \(T\)，负采样数为 \(b\)。

    论文证明，当随机游走长度趋于无穷且使用负采样时，DeepWalk 隐式分解的矩阵 \(M\) 可写为： [ M = \log \left( \frac{\text{vol}(G)}{b} \left( \frac{1}{T} \sum_{r=1}^T P^r \right) D^{-1} \right). ] 矩阵元素 \(M_{ij}\) 表示节点 \(i\) 与 \(j\) 的关联强度，它综合了 \(T\) 步内从 \(i\) 到 \(j\) 的平均转移概率，并通过度矩阵做了归一化。该形式与自然语言处理中的点间互信息（PMI）矩阵高度相似。

    类似地，LINE 的二阶邻近目标等价于分解 [ M_{\text{LINE}} = \log \left( \frac{\text{vol}(G)}{b} P D^{-1} \right), ] 这正是 DeepWalk 在窗口大小 \(T=1\) 时的特例。对于采用二阶随机游走的 node2vec，其等价矩阵涉及由 \(p,q\) 参数化后的二阶转移张量，但同样是某个闭式矩阵的分解。

    4.2 与谱图理论的联系

    进一步，可以将 DeepWalk 矩阵写成对称形式： [ S = \frac{\text{vol}(G)}{b} D^{-1/2} \left( \frac{1}{T} \sum_{r=1}^T (D^{-1/2} A D^{-1/2})^r \right) D^{-1/2}. ] 矩阵 \(D^{-1/2} A D^{-1/2}\) 正是归一化邻接矩阵，其特征值分解可以联系到归一化图拉普拉斯 \(L = I - D^{-1/2} A D^{-1/2}\)。中间的求和项 \(\frac{1}{T} \sum_{r=1}^T \Lambda^r\) 相当于一个低通滤波器：随机游走的高阶幂次使较小特征值（图的高频噪声）迅速衰减，而保留反映社区结构的大特征值。因此，DeepWalk 等算法本质上是隐式地对图的归一化拉普拉斯矩阵进行低通谱变换，从而保留了宏观的拓扑相似性。

    4.3 NetMF 算法

    既然这些方法都在隐式分解矩阵，那么可以直接显式计算该矩阵并进行分解，从而避免随机采样的不稳定。NetMF 的做法是： 1. 构建目标矩阵 \(M\)，直接计算 [ M = \log \left( \max \left( \frac{\text{vol}(G)}{b T} \sum_{r=1}^T P^r D^{-1}, 1 \right) \right). ] 截断小于 1 的值是为了保证对数非负。 2. 对 \(M\) 进行奇异值分解（SVD），保留前 \(d\) 个奇异值和奇异向量，作为节点的低维嵌入。

    对于大规模图，直接计算矩阵幂和存储稠密矩阵代价过高。NetMF 提出近似方案：利用归一化邻接矩阵的稀疏性，通过 Arnoldi 算法求出前 \(h\)（\(h \ll n\)）个特征对 \(\{(\lambda_i, u_i)\}\)，然后近似重构求和项： [ \sum_{r=1}^T P^r D^{-1} \approx D^{-1/2} \left( U_h \left( \frac{1}{T} \sum_{r=1}^T \Lambda_h^r \right) U_h^\top \right) D^{-1/2}. ] 这大幅降低了时间和空间开销，使算法能够应用到较大规模的图上。

    4.4 统一视角的意义

    NetMF 的工作将看似割裂的“随机游走+深度学习”方法纳入经典的谱图理论与矩阵分解框架，消除了这些算法的黑盒感。它也解释了为何 DeepWalk 和 node2vec 能有效保持图的社区结构。从应用角度看，显式矩阵分解无需采样，结果更稳定；但受限于稠密矩阵存储，扩展到超大规模图时仍需要进一步的稀疏化技术（如后续的 NetSMF）。在理论层面，这一统一为图神经网络的设计提供了清晰的基础。

    5. 总结

    DeepWalk 开创性地将随机游走和 Skip-gram 引入图表示学习，将图结构转化为线性序列进行建模。node2vec 通过引入返回参数 \(p\) 和进出参数 \(q\) 实现有偏随机游走，可以更精细地捕捉同质性与结构等价性。NetMF 则从理论上证明，这些方法本质上都在隐式地分解一个由图的转移概率决定的闭式矩阵，并与谱图理论建立了直接联系。

    这三种方法构成了一个递进的理论体系：从启发式采样到可控探索，再到统一的矩阵分解视角。理解它们的原理和联系，有助于把握图嵌入方法的发展脉络，并为学习后续更复杂的图神经网络奠定基础。
